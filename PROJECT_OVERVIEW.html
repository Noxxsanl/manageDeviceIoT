IoT Device Manager — RBAC

Hệ thống quản lý thiết bị IoT end-to-end: từ firmware ESP32 thu thập cảm biến, qua dual MQTT broker, backend TypeScript xác thực HMAC, đến dashboard Next.js với phân quyền RBAC đầy đủ. Tài liệu này tổng hợp toàn bộ những gì đã được xây dựng.

Bảo Mật

HMAC-SHA256 · RBAC · Auto-block · JWT · Rate limiting · Helmet · Timing-safe

2-Layer HMAC Authentication

Mỗi message đi qua 2 lớp xác thực độc lập: sensor ký payload, gateway ký lại trước khi gửi backend.

hmacService.ts · 2-Layer HMAC verification chain
%%{init:{"theme":"dark","themeVariables":{"primaryColor":"#1c2030","primaryTextColor":"#e2e8f0","primaryBorderColor":"#2a2f4a","lineColor":"#64748b","edgeLabelBackground":"#1c2030"}}}%% sequenceDiagram participant SN as 🌡️ Sensor Node participant B1 as 📶 Broker 1 participant GW as 📡 Gateway Node participant B2 as 📶 Broker 2 participant BE as ⚙️ Backend SN->>SN: HMAC_SHA256(secret_key, "sn_id:timestamp") SN->>B1: publish local/sensors/sn1/data
{sn_id, sn_timestamp, sn_hmac, temperature, humidity} B1->>GW: message received GW->>GW: verify sn_hmac
check timestamp ±300s GW->>GW: HMAC_SHA256(gw_secret, "gw_id:timestamp") GW->>B2: publish gateway/gw1/data
{gw_id, gw_timestamp, gw_hmac, [sensor_payload]} B2->>BE: mqttDataService receives message BE->>BE: Layer 1: verify gw_hmac BE->>BE: Layer 2: verify sn_hmac BE->>BE: INSERT sensor_data BE->>BE: UPDATE last_seen, fail_count=0

Cơ chế bảo vệ

  • crypto.timingSafeEqual() — chống timing attack
  • Timestamp window ±300s — chống replay attack
  • Mỗi device có secret_key riêng trong DB
  • Secret key chỉ trả về 1 lần khi register
  • HMAC algorithm: SHA-256 (không thể giả mạo)

Điều kiện tăng fail_count

  • TIMESTAMP_EXPIRED — timestamp lệch > 300s
  • HMAC_MISMATCH — chữ ký không khớp
  • NOT_FOUND — không thay đổi fail_count
  • DEVICE_BLOCKED — không thay đổi fail_count
  • Auth success → reset fail_count = 0

RBAC — Phân Quyền 3 Tầng

RoleQuyềnHạn chế
admin Toàn quyền: CRUD users, devices, xoá audit log, thay đổi mọi trạng thái Không thể tự xoá bản thân. Không thể xoá tài khoản admin khác.
operator Đăng ký thiết bị, quản lý devices (status, xoá), xem audit log Không quản lý được users. Không xoá được tài khoản.
viewer Xem danh sách devices, dashboard stats, lịch sử dữ liệu Chỉ đọc. Không thể thực hiện bất kỳ thao tác ghi nào.
